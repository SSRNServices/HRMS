# Tech Stack – HRMS SaaS

## 1. Backend
- Frappe Framework (Python)
- MariaDB / PostgreSQL
- Redis (caching + queue)

## 2. Frontend
- Next.js (React)
- Tailwind CSS
- Axios / Fetch API

## 3. Mobile
- Flutter (recommended)
OR
- React Native

## 4. Infrastructure
- Docker (recommended)
- Nginx
- Supervisor
- Cloud (AWS / GCP / DigitalOcean)

## 5. Dev Tools
- GitHub
- CI/CD (GitHub Actions)
- Postman / Swagger

## 6. Authentication
- JWT
- OAuth (optional)
- MFA

## 7. Realtime
- Socket.io (Frappe built-in)

## 8. Storage
- S3 / MinIO (documents)

## 9. Analytics
- Built-in reports (Frappe)
- Optional: Metabase / Superset

## 10. Monitoring
- Prometheus + Grafana (optional)
